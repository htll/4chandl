4chandl
=======

4chan image download script

Usage: 4chandl.py [Options] thread_url

Downloads all images from a 4chan thread

Options:
  -h, --help            show this help message and exit
  -f OUTFOLDER, --folder=OUTFOLDER
                        Location where to download iamges
  -s SLEEPTIMER, --sleep=SLEEPTIMER
                        How long to wait between each image download
